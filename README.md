# AI Study Planner **AI-powered Study Planner** web app built with **Python** and **Flask**.
This project helps students plan their daily study routine based on available time, subjects, and energy levels.
It generates a smart, prioritized schedule to optimize study efficiency.
--- ## Features - Input your **daily routine**: wake-up time, sleep time, school & tuition hours, meals, travel, and breaks.
--- Add **subjects** you want to study. 
--- **AI-generated study schedule** prioritizing subjects based on urgency and difficulty.
---Simple and clean **web interface** built with Flask and HTML. - Save daily plans locally in JSON format.
--- ## Tech Stack - Python 3.x - Flask - HTML/CSS - Jinja2 Templates 
--- ## Installation

1. Clone the repository:
bash
git clone https://github.com/yourusername/AI-Study-Planner.git
Navigate to the project folder:

cd AI-Study-Planner


2. Install dependencies:
pip install flask


3. Run the app:
python app.py


4. Open in your browser:
http://127.0.0.1:5000


5. Usage

-->Fill in your daily routine and subjects in the web form.
-->Click Generate Plan to get your AI-generated study schedule.
-->Follow the suggested schedule for efficient study.
